ASCII Math Experiment

Initial, naieve, discussable example of using an ascii-math custom element which uses an ascii-math to mathml converter to create mathml in the shadow tree and style the generated operators.

Simple example

Code sample: 
<ascii-math>sum_(i=1)^n i^3=((n(n+1))/2)^2</ascii-math>
+
renders as: sum_(i=1)^n i^3=((n(n+1))/2)^2
